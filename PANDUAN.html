▶ YT Downloader

Download video & playlist YouTube dengan mudah — rename massal, pilih format, simpan ke folder sendiri.

Persyaratan

SoftwareKeteranganStatus
Python 3.10+ Runtime utama aplikasi Wajib
FFmpeg Diperlukan untuk MP3 dan video HD (1080p+) Direkomendasikan
Koneksi internet Untuk download dari YouTube Wajib
ℹ
Tanpa FFmpeg, download masih bisa berjalan untuk format video standar (360p–720p). FFmpeg diperlukan untuk MP3 dan 1080p ke atas.

Instalasi

Install Python

Download dari python.org dan jalankan installer.
Pastikan centang "Add Python to PATH" saat instalasi.

Install FFmpeg (Direkomendasikan)

Buka CMD atau PowerShell, ketik:

winget install ffmpeg

Atau download manual dari gyan.dev/ffmpeg/builds lalu tambahkan ke PATH.

Jalankan Aplikasi

Double-click file start.bat. Aplikasi akan:

  • Install dependencies otomatis (flask, yt-dlp)
  • Membuka browser ke http://localhost:5000

Nama file bisa diedit langsung di kolom track. Klik ↺ untuk reset ke judul asli.

Pilih MP4/MP3/WebM dan kualitas yang diinginkan, klik Download.

YouTube Radio / Mix

Link YouTube Radio/Mix (list=RD... atau start_radio=1) adalah playlist auto-generated yang bisa berisi ratusan hingga ribuan track.

⚠
Jangan fetch tanpa batas — bisa lama sekali dan hasilnya terlalu banyak. Selalu gunakan fitur Batas track.

Saat URL Radio/Mix terdeteksi, aplikasi otomatis menampilkan peringatan dan menyarankan batas 25 track. Angka ini bisa diubah sesuai kebutuhan.

FAQ & Troubleshoot

❌ Error: ffprobe and ffmpeg not found

FFmpeg belum terinstall atau tidak ada di PATH. Jalankan perintah ini di CMD:

winget install ffmpeg

Setelah selesai, tutup dan buka ulang start.bat.

❌ Error saat fetch playlist — "could not find..."

URL mungkin salah atau video bersifat privat/region-locked. Pastikan URL bisa dibuka di browser terlebih dahulu.

⚠ Download berhasil tapi file tidak ditemukan

Cek folder yang ditampilkan di bagian Simpan dalam subfolder di halaman utama. Klik ikon ⚙ untuk melihat folder aktif.

⚠ Playlist fetch lama sekali

Kemungkinan playlist sangat panjang atau koneksi lambat. Untuk Radio/Mix, gunakan fitur Batas track. Untuk playlist biasa yang panjang, bisa isi batas track juga sebelum fetch.

🔄 Update yt-dlp jika video tidak bisa didownload

YouTube sering update. Jika tiba-tiba banyak error, update yt-dlp:

pip install -U yt-dlp
🌐 Akses dari perangkat lain di jaringan yang sama

Buka app.py, ubah baris terakhir dari app.run(...) menjadi:

app.run(debug=False, host='0.0.0.0', port=5000, threaded=True)

Lalu akses dari perangkat lain via http://[IP-komputer]:5000.
